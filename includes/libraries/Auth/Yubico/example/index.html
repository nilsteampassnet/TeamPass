Instructions for using the Yubikey

1. Insert your Yubikey in the USB-port with the USB-contact facing upward. For PC, the first time you use the Yubikey, wait until Windows has detected the device. For Mac OS X, choose any keyboard to use. Make sure that the touch-button shines with a steady light.

2. Place the input cursor in the Yubikey input field.

3. Place your fingertip on the button, hold for a second and the Yubikey sends your unique ID and a time variant pass code to the server.
